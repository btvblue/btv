Loading

로딩은 작업에 대한 대기가 필요할 때 사용합니다.
페이지 전환이 지연될 때 사용되며, 요청이 처리 되고 있음을 사용자에게 알립니다.
이를 통해 사용자들이 작업의 진행 상태를 이해하고 기다림에 대한 양해를 가지게 됩니다.
