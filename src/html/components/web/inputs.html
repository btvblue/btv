Inputs

사용자로부터 정보나 데이터를 입력받는 데 사용됩니다.

  • 일반적으로 텍스트, 숫자, 날짜 등의 데이터를 입력받을 수 있는 입력 필드를 포함합니다.

Additional

인풋은 사용자와의 상호작용을 위한 기능을 추가적으로 제공할 수 있습니다.
입력 필드에 자동 완성, 드롭다운 메뉴, 슬라이더 등의 기능을 함께 사용하여
사용자가 편리하게 데이터를 입력하고 선택할 수 있도록 도와줍니다.
