Dock

독 메뉴는 서비스의 일부를 차지하는 고정된 위치에 위치한 메뉴입니다.
사용자에게 중요한 앱에 쉽고 빠르게 접근할 수 있는 방법을 제공합니다.

  • 사용자에게 항상 일관된 공간에서 메뉴에 접근할 수 있도록 해줍니다.
  • 각 메뉴 항목은 앱을 나타내며, 사용자가 해당 항목을 선택하면 해당 앱이 실행됩니다.
