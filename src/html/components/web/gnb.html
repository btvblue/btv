GNB

Global Navigation Bar의 약자로, 서비스의 가장 기본적인 내비게이션을 담당하는 메뉴입니다.
서비스의 정체성을 전달하는 역할로 사용자 동선을 안내와 상위 레벨 탐색을 위해 제공되는 컴포넌트입니다.

  • GNB는 웹사이트의 모든 페이지에서 일관된 네비게이션을 제공합니다.
  • 사용자가 사이트 내에서 쉽게 이동하고 기능에 접근할 수 있도록 도와줍니다.

Layout

GNB는 여러 수준의 하위 메뉴를 포함할 수 있습니다.
이는 사용자가 상위 메뉴에서 선택한 옵션에 따라 하위 메뉴가 동적으로 나타날 수 있는 기능을 의미합니다.
하위 메뉴는 사용자에게 더 많은 선택 옵션을 제공하고, 네비게이션의 깊이를 늘릴 수 있습니다.
