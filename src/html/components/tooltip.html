Tooltip

Tooltip은 사용자가 익숙하지 않은 구조나 기능에 대해 도움을 줄 때 주로 사용이 되며, 정보의 정확한 조작을 유도할 때도 표시합니다. 특정 이벤트 프로모션이 진행 되는 경우 부가설명을 위해 사용하기도 합니다.

Placement

Tip 위치

Tip의 위치는 타겟 요소의 배치에 따로 상/하단으로 제공 될 수 있으며 타겟이 되는 요소를 가리키기 위하여 유동적으로 배치합니다.

Bubble Tooltip

Tip의 위치는 항상 타겟이 되는 요소를 가까이 가리키고 있어야 합니다.
