Switch

Switch는 상호간의 배타적인 상태의 전환이 필요할 때, 기능을 ON/OFF 하기 위해 사용합니다. 사용자가 선택하지 않아도 기본적으로 선택되어 있는 Default Option을 제공합니다.

Types

General switch

기본적으로 사용되는 Switch로, ON/OFF의 기능을 수행합니다.
사용자의 시각적 이해를 돕기위해 선택된 Option에 Color로 강조합니다.
