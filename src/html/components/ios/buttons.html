1. 시선의 흐름에 따른 배치

List를 포함한 다른 Contents와 함께 사용할 경우 시선의 흐름에 따라 배치 및 사용을 권장합니다.

2. Button 배치

Button의 선택지를 2개로 제공할 때, 긍정을 의미하는 Button은 좌측에 위치하고 부정을 의미하는 Button은 우측에 배치하는 일관된 형태를 권장합니다. 우선 순위가 높은 순으로 좌측에서 우측으로 제공합니다.

*구매화면의 결제 Button은 예외적으로 긍정의 Button이 우측에 사용됩니다.

@@include("../component/image_content_box.html", { "mode" : "", "image" : "components/buttons/buttons_image_018", "desc" : null })

3. 화면 Layout에 따른 정렬 방식

Button 정렬 방식은 화면의 Layout에 따라 달라질 수 있습니다.
기본적으로 Icon과 Text이 같이 제공될 경우 영역 내 중앙 정렬을 권장하며 화면 정렬 방법에 따라 좌측 정렬으로 사용 가능합니다.
