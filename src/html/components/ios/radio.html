Radio Button

Radio Button은 단일 Option만 선택할 수 있는 경우 사용합니다. 같은 List 내 다른 Radio Button 선택 시 기존에 선택되어 있었던 Radio Button이 선택 해제 되며, 항상 Default Option을 제공합니다.

States

Radio Button의 상태별 정의는 리모컨의 포커싱에 따라 focus/normal 상태로 크게 분류되며,
상태에 따라 select/unselect 로 분류됩니다. 선택할 수 없는 Option의 경우는 dimmed 상태의 Radio Button을 사용합니다.
