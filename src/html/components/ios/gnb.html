GNB

서비스를 구성하는 가장 기본적인 뼈대이자 정보구조이며, 사용자 동선을 안내하는 중요한 서비스 구성요소입니다.
GNB는 서비스의 정체성을 전달하는 역할로 상위 Level 탐색에 사용됩니다.

Types

General Home

Select 된 Menu의 경우 좌측에 Line 노출하여 사용자의 현위치를 알립니다.

1Depth/2Depth

GNB가 Focus되어 열린 상태로 관련 Icon과 Text를 함께 노출합니다.
1 Depth에서 추가적으로 하위 Menu List가 있을 경우 2 Depth를 노출하며, 해당 Category의 관련 List를 제공합니다.
