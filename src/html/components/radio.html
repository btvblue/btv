Radio Button

Radio Button은 단일 Option만 선택할 수 있는 경우 사용합니다. 같은 List 내 다른 Radio Button 선택 시 기존에 선택되어 있었던 Radio Button이 선택 해제 되며, 항상 Default Option을 제공합니다.

A General Select / Unselect Radio Button

Anatomy

  • 1. Container

    Radio Button의 형태를 나타냅니다.

  • 2. Icon

    Inner Circle의 존재 유무로 선택 여부를 판단 할 수 있습니다.

  • 3. Text

    해당 Text가 우측에 배치됩니다.

Types

General Radio Button

기본형의 Radio Button으로, Text와 함께 노출합니다.

Do

항상 Default Option 선택값을 제공하여 무조건 선택해야하는 Option임을 사용자가 인지할 수 있어야합니다.

Don't

Radio Button Option의 항목 안의 항목으로 중첩하여 사용하지 않습니다.

States

Radio Button의 상태별 정의는 리모컨의 포커싱에 따라 focus/normal 상태로 크게 분류되며,
상태에 따라 select/unselect 로 분류됩니다. 선택할 수 없는 Option의 경우는 dimmed 상태의 Radio Button을 사용합니다.

Do

선택 가능한 List의 Option들은 용도에 따라 알맞게 그룹화 하여 사용합니다.

Don't

그룹 간 List의 간격을 붙여서 사용하지 않습니다.

Usage

Vertical Radio Button

기본형의 Radio Button을 Text와 함께 세로로 노출합니다. 주로 디테일한 정보의 Option을 나열할 때 사용합니다.

Horizontal Radio Button

기본형의 Radio Button을 Text와 가로로 노출합니다. 주로 Option의 내용이 간단 명료 할 때 사용합니다.

Don't

Radio Button 배열 시 정보의 순서를 랜덤으로 배치하지 않습니다.

Image Radio Button

사용자에게 Option의 더 명확한 설명을 돕기 위해 Image가 Radio Button 컴포넌트 상단에 함께 노출됩니다.
