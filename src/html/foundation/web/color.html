Color

SK B tv Color

B tv UI의 주요 색상은 브랜드 컬러인 T Blue를 기반으로 제작했습니다.
토널 팔레트를 활용하여 색상을 정의하고 포커스 컬러는 웹 접근성(WCAG)을 고려하여
밝은 화면과 어두운 화면에서 동일하게 사용할 수 있는 토널 팔레트의(5C5FFF)를 사용합니다.

Surface Color

사용자 인터페이스 요소의 배경의 색상을 나타내는 개념입니다.
UI 디자인에서 요소의 표면에 적용되는 색상을 의미하며 주로 배경 패널, 카드, 버튼, 입력 필드 등의 요소에 적용됩니다.
UI의 배경 느낌과 분위기를 결정하는데 중요한 역할을 합니다.
B tv의 경우 다양한 환경에서 노출되는 경우가 많기 때문에, 밝은 환경과 어두운 환경에 모두 적합한 색상을 사용해야 합니다.
